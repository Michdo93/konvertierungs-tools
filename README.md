# 🔧 Konvertierungs-Tools

Browser-basierte Konvertierungs-Suite – kein Backend, keine Dateiuploads an Server.

🔗 **Live:** `https://<dein-username>.github.io/konvertierungs-tools/`

## Tools

| Tool | Funktionen |
|------|-----------|
| [Text-Tools](pages/text.html) | Groß-/Kleinschreibung, Umkehren, Slug, camelCase, snake_case, Sortieren, Deduplizieren, Zählen |
| [Farb-Konverter](pages/farbe.html) | HEX ↔ RGB ↔ HSL ↔ HSV, Luminanz, Komplementär, Graustufe, Farbpicker, Palette |
| [Hash-Generator](pages/hash.html) | SHA-1, SHA-256, SHA-384, SHA-512 via WebCrypto (lokal!), Datei-Hash |
| [Base64 / URL / HTML](pages/base64.html) | Base64 Encode/Decode, Datei→Base64, URL-Encoding, HTML-Entities |
| [JSON / CSV](pages/json-csv.html) | JSON formatieren/minifizieren/validieren/sortieren, JSON↔CSV |
| [Datum & Zeit](pages/datum.html) | Unix-Timestamp↔Datum, ISO 8601, Datumsdifferenz, Zeitzonenübersicht |

## Deployment
```
Settings → Pages → Source: main / root
```
